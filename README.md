## Overview / Обзор

AXON is a Windows application for YouTube, Discord and Telegram.

AXON - Windows-приложение для YouTube, Discord и Telegram.

## Install / Установка

1. Download `AXON.exe` from [Releases](https://github.com/awkcrud/AXON/releases/latest)
2. Run as Administrator
3. Sign in with your account
4. Start a strategy for YouTube / Discord, or enable the Telegram proxy

Close VPN and other proxy clients before starting.

1. Скачай `AXON.exe` из [Releases](https://github.com/awkcrud/AXON/releases/latest)
2. Запусти от имени администратора
3. Войди в аккаунт
4. Запусти стратегию для YouTube / Discord или Telegram-прокси

Перед стартом останови VPN и другие прокси.

## Updates / Обновления

The app checks this site for new versions. Silent notice is soft; full details are in the update dialog from the menu.

Приложение сверяет версию с этим сайтом. Тихое уведомление мягкое; полное описание - в окне обновлений из меню.
